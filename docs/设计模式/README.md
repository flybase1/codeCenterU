# 设计模式

一开始是看的大话设计模式，初步学习了一些设计模式，然后跟着鱼皮做了聚合搜索，用到了门面模式，适配器模式等等，虽然知道里面，还是无法独立
想到这些模式如何搭配使用，一方面是没掌握好对应的设计模式，另一方面自己接触的实战场景也不多，所以决定重新学习设计模式，这里我推荐
